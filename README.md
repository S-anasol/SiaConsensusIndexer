## Installation

### Web
- ``git clone https://github.com/S-anasol/SiaConsensusIndexer.git``
- import database.sql
- Edit mysql config https://github.com/S-anasol/SiaConsensusIndexer/blob/master/update.php#L111
- Edit siad address (localhost:9980)
- run ``php update.php``
- Done!
