## Installation

### Web
- ``git clone https://github.com/SiaHubLab/SiaConsensusIndexer.git``
- import database.sql
- Edit mysql config https://github.com/SiaHubLab/SiaConsensusIndexer/blob/master/update.php#L111
- Edit siad address (localhost:9980)
- run ``php update.php``
- Done!
